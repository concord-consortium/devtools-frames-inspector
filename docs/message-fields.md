# Message Field Reference

This document describes the fields shown in the Context tab when viewing a message.

## Message ID

Unique identifier for this message.

Generated by the extension when the message event fires in the page context.

---

## Timestamp

When the message was received.

Captured from Date.now() when the message event fires in the page context.

---

## Size

Size of the message data when serialized.

Calculated from JSON.stringify(event.data).length.

---

## Buffered

Whether this message was captured before the panel connected.

Messages sent before DevTools opens are buffered if the page was opened from a monitored tab.

---

## Target URL

The full URL of the window that received this message.

Obtained from window.location.href of the receiving frame.

---

## Target Origin

The origin of the window that received this message.

Obtained from window.location.origin of the receiving frame.

**Filter:** `target:example.com`

---

## Target Title

The document title of the receiving window.

Obtained from document.title of the receiving frame.

---

## Target Frame

The Chrome frame ID of the receiving window.

Obtained from sender.frameId in the background script.

---

## Target Frame Error

Error that occurred when trying to get frame information.

Usually indicates the content script could not access frame APIs.

---

## Source Type

The relationship between the sender and receiver windows.

Determined by comparing event.source to window.parent, window.opener, and child frames.

**Filter:** `sourceType:parent`

---

## Source Origin

The origin of the window that sent this message.

Obtained from MessageEvent.origin. May be "null" for data: or file: URLs.

**Filter:** `source:example.com`

---

## Source Frame

The Chrome frame ID of the sending window.

Only currently supported when the message comes from window.parent. Obtained from chrome.webNavigation.getFrame in the background script.

---

## Source iframe src

The src attribute of the iframe element that sent the message.

Only available for child sources. Read from the iframe element in the parent document.

---

## Source iframe id

The id attribute of the iframe element that sent the message.

Only available for child sources. Read from the iframe element in the parent document.

---

## Source iframe path

CSS selector path to the iframe element that sent the message.

Only available for child sources. Generated from the iframe element position in the DOM.

---
